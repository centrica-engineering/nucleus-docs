The `ns-datepicker` wraps the `ns-inputter` component. Guidelines defined for the `ns-inputter` should also be adhered to when using this component. See [`<ns-inputter>`](/components/ns-inputter) documentation.

## Guidance

:::do
- Use the `standard` type for picking a date from a range.
- Use the `standard` type for picking a date that the user might not be sure of (e.g. Next Friday).
- Use the `masked` type for picking a date that is well known to the user (e.g. Date of birth).
:::

:::dont
- Change the date format.
:::
