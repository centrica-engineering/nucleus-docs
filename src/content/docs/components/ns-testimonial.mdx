## Guidance

:::do
- Use a single sentence.
- Use `<b>` to highlight key phrases.
- Use positive feedback.
- Use multiple quotes in a row.
:::

:::dont
- Use multiple paragraphs.
- Link to an external website.
- Include anything negative, rude or which may be misconstrued.
- Include any personal information beyond first name or social media handle.
:::
