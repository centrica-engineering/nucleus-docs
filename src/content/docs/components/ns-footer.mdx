## Overview

<CompDetails name={componentName} />

:::note
This is the UI component for producing a customised footer. If you're looking for the default British Gas footer containing the links, please use the [`<nsx-footer>`](/components/nsx-footer) component.
:::

The footer is the last component on the page and holds links and copyright notices that are legally required. It also contains information about the company.

### Examples

<ExampleViewer type="standard" />

## Guidance

:::do
- Always situate the footer at the bottom of a page.
- Include `<ns-footer>` after the closing `</main>` element.
:::

:::dont
- Make a unique, custom footer if it is to be part of the customer-facing website.
:::
