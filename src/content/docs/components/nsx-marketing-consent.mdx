## Guidance

:::do
- Always keep the same options.
- Use at the end of a journey that captures contact information used for marketing purposes.
- Always use inside of an `ns-form` component.
- Use only once per journey.
:::

:::dont
- Reorder the options.
- Use when collecting contact information that will be used solely for a service offering e.g. Booking an engineer.
- Have any of the options pre-checked.
:::
