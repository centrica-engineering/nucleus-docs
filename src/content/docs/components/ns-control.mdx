## Overview

<CompDetails name={componentName} />

`ns-control` allows users to take actions or make choices with a single interaction. These buttons can have text visible or hidden if the icon is enough to convey meaning.

## Guidance

:::do
- Use short text to describe the action being taken.
- Use for lower emphasis actions, such as opening a modal.
- Relate the action to the adjacent content.
:::

:::dont
- Try to use within an `ns-cta` to render an icon. Use the `ns-cta`'s icon attribute instead.
- Make it overly wordy.
- Use for downloads â€“ use [`<ns-download>`](/components/ns-download) instead.
- Use more than 4 `ns-controls` at once as it can cause cognitive load and usability issues. 
:::
