## Overview

<CompDetails name={componentName} />

`ns-control` allows users to take actions or make choices with a single interaction. These buttons can have text visible or hidden if the icon is enough to convey meaning.

<ExampleViewer type="standard" />

## Guidance

:::do
- Use to control the user interface.
- Relate the label to the icon in the control.
- Use short text to describe the control.
:::

:::dont
- Make the label overly wordy.
- Try to use within an `ns-cta` to render an icon. Use the `ns-cta`'s icon attribute.
- Use for downloads â€“ use [`<ns-download>`](/components/ns-download) instead.
- Use more than 4 `ns-control` components together as it can cause cognitive load.
:::
