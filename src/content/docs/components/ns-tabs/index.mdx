## Purpose

> **Use to provide the organisation of content separated within the same context.**

The `ns-tabs` component separates related content into clearly labelled sections. It comprises multiple `ns-tab` components.

`ns-tab` is a singular tab that dictates what the area is about. It is the button that can be clicked to change the content in `ns-tabs` and is never used in isolation. It shows the user what content there is with a heading and an optional icon.
