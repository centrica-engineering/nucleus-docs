## Overview

<CompDetails name={componentName} />

Illustrations simplify products and services to users in a friendly manner. Illustrations always accompany the content.

### Examples

All available illustrations can be seen on the [Illustrations](/guidelines/assets#illustrations) section of the Assets page.

<ExampleViewer type="standard" />

## Guidance

:::do
- Use them to convey depth.
- Use them to differentiate or define the content.
- Use them to help signpost.
- Check existing usage.
:::

:::dont
- Make the illustrations small.
- Use on their own as the primary visual element.
- Use instead of icons. They have a different purpose. (See [`<ns-icon>`](/components/ns-icon).)
- Alter the colours or apply visual effects.
:::
