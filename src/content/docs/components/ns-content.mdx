The `ns-content` component's placement is inside other components. It helps to add consistency to the styling of typography, improving how it interacts with the other components.

## Guidance

:::do
- Use clean HTML.
- Set the [`spacing`](/components/ns-content#specification-attribute-spacing) attribute to complement the content it contains.
:::

:::dont
- Use additional `<div>` tags to wrap content - it will break the spacing.
:::

#### Specification notes

##### Anonymous slot

To display correctly, `<ns-content>` relies on clean HTML in the anonymous slot. Using additional components or HTML elements not listed in the [permitted tags](#specification-slots) will break the styling and will be susceptible to breaking when the component is updated in future releases.

##### `spacing` options

| Spacing | Space value | 
| :--- | :--- |
| `compact` | `0.25em` |
| `cozy` | `0.5em` |
| `normal` | `1.0em` |
| `loose` | `1.5em` |
