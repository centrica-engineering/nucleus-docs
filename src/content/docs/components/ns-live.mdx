## Guidance

:::do
- Use to wrap small pieces of dynamic content.
- Keep the content simple.
- Leave the old content in place whilst busy.
- Remove the busy attribute and update content at the same time.
:::

:::dont
- Wrap large parts of the UI.
- Place inside text elements (`<p>`, `<h1>` - `<h6>`).
- Use for media.
:::
