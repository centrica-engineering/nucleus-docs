Use to allow users to filter a list of items based on a set of criteria. It is commonly used in product listings and other scenarios to narrow down a large list of items.

## Guidance

:::do
- Group related filters logically (e.g., by category or relevance).
- Provide clear, descriptive headings for each filter group.
- Highlight default filters only if they clearly align with the userâ€™s likely needs or preferences.
:::

:::dont
- Use filters to hide essential information from users.
- Overload the page with too many filter options visible at once.
- Nest filters within other filters, creating complex navigation.
- Use vague or unclear labels that confuse users.
- Show filter options that result in 0 results without providing feedback or alternatives.
:::
