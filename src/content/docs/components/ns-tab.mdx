## Overview

<CompDetails name={componentName} />

The `ns-tab` is the button that changes the content in the `ns-tabs` when clicked. It shows the user relevant content with a heading and an optional icon.

:::caution
This component can only be placed within `ns-tabs`. Refer [`<ns-tabs>` Specification](/components/ns-tabs#specification-slots) for usage.
:::

### Examples

<Tabs>
  <TabItem label="Standard">
    <ExampleViewer type="standard" />
  </TabItem>
  <TabItem label="Icon">
    <ExampleViewer type="standard-icon" />
  </TabItem>
</Tabs>

## Guidance

:::do
- Always add heading text.
:::

:::dont
- Add an icon to one tab without adding one to all other tabs.
- Add more than two words to the heading.
- Mix different types of [icons](/guidelines/assets#icons) (i.e. solid, outline, functional).
:::
