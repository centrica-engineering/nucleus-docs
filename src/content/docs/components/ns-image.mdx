## Guidance

:::do
- Resize your image to the correct dimensions for the component.
- Use JPEG and WebP image format.
- Use brand approved images.
- Think about the image responsive behaviour.
- Use images hosted in CMS.
:::

:::dont
- Use images as the primary content.
- Put text inside the image.
- Use a white background in images.
- Use media type as alt text such as 'an image'.
:::
