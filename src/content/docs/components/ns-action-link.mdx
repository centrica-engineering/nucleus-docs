## Guidance

:::do
- Make it short and actionable.
- Add illustrations.
- Relate the action to the content it sits beside.
:::

:::dont
- Make it vague.
- Make it overly wordy.
- Treat as a button.
- Use more than 24 characters.
- Link to outside urls.
- Use for downloads, use [`<ns-download>`](/components/ns-download).
:::
