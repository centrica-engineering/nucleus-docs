## Guidance

:::do
- Make it short and actionable.
- Add a [`decoration`](#specification-attribute-decoration).
- Relate the action to adjacent content.
:::

:::dont
- Make it vague.
- Make it overly wordy.
- Treat it as a button.
- Use more than 24 characters.
- Link to outside URLs.
- Use for downloads, use [`<ns-download>`](/components/ns-download) instead.
:::
