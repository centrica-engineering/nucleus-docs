## Overview

<CompDetails name={componentName} />

Use the `ns-form-group` component instead of the usual HTML fieldset. The `ns-form-group` extends the layout for `ns-form`.

### Examples

<ExampleViewer type="standard" />

## Guidance

:::do
- Provide a `legend` attribute with value.
- Only use inside the `ns-form` component.
:::

:::dont
- Group unrelated `ns-inputter`s.
- Nest `ns-form-group`s unnecessarily, as this can lead to confusion.
:::

## Implementation

### Placement

<Placement name={componentName} />

:::note
When adding radio buttons and checkboxes to `ns-inputter` inside an `ns-form-group`, its heading size will match the size of all labels from other `ns-inputter`. This helps keep a logical visual hierarchy in a form.
:::
